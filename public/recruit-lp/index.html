FAQ

採用LP制作の流れはどのようになりますか？

add

採用LP制作の流れは大きく分けて5ステップです：①ヒアリング（採用課題や求める人材像の確認）、②企画・設計（ターゲット分析とメッセージ設計）、③デザイン制作、④コーディング（レスポンシブ対応）、⑤公開・運用です。企画から制作、公開まで一貫してサポートいたします。

納品までの期間はどのくらいですか？

add

スタンダードプランの場合、ヒアリング後から最短3営業日以内に納品いたします。プレミアムプラン以上は、動画撮影や追加コンテンツの制作が必要なため、別途納期をご相談させていただきます。緊急の採用活動が必要な場合でも、可能な限り対応させていただきます。

料金以外に発生する費用はありますか？

add

料金プランに記載されている金額の他に、サーバー費用やドメイン取得費用が別途発生します。また、オプションでご希望の場合は、写真・動画の追加撮影費、広告運用費なども発生します。すべて事前にお見積りをご提示し、ご了承いただいた上で進めます。

採用LPの効果はどのように測定されますか？

add

採用LPの効果測定は、アクセス数、滞在時間、フォーム到達率、応募率（コンバージョン率）などの指標を通じて行います。Googleアナリティクスなどの解析ツールを導入し、ユーザーの行動データを詳細に分析します。プレミアムプラン以上では、月次レポートで効果測定結果と改善提案をご提供しています。

どんな業種や企業規模に対応していますか？

add

製造業、建設業、IT・情報通信、小売業、サービス業など、幅広い業種に対応しています。特に採用に苦戦されることが多い中小企業やレガシー産業に強みを持っていますが、スタートアップから大企業まで、あらゆる企業規模のお客様にサービスを提供しています。

LP制作後のサポートはありますか？

add

はい、LP制作後もサポートをご提供しています。スタンダードプランでは、初回テキスト修正1回を含みます。プレミアムプラン以上では、3ヶ月間の月次レポーティングとテキスト更新（毎月1回）、オンライン相談会などのサポートが含まれています。継続的なメンテナンスや更新が必要な場合は、別途サポートプランもご用意しています。

どのような採用課題の解決に効果がありますか？

add

以下のような採用課題の解決に効果があります：
・応募数が少なく、人材を確保できない
・採用のミスマッチによる早期離職が多い
・大手企業に人材を奪われる
・人材紹介会社への依存度が高く、採用コストがかさむ
・自社の魅力や特徴を効果的に伝えられていない
・採用ブランディングを強化したい
